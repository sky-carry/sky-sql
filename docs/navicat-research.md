（目标：为复刻类 Navicat 桌面数据库管理工具 SkySQL 提供产品需求输入。调研日期：2026-06-10）

## 1. 支持的数据库类型与连接方式

### 1.1 数据库支持矩阵

Navicat Premium 17 是"一个客户端连接所有数据库"的定位，单一应用内可同时建立多种数据库连接：

| 数据库 | 说明 |
|---|---|
| MySQL | 最核心支持，兼容 Amazon RDS/Aurora、Azure、Google Cloud 等云 MySQL |
| MariaDB | 与 MySQL 同级支持 |
| PostgreSQL | 8.4+，含云 PG、Amazon Redshift（兼容协议） |
| Oracle | 支持 Basic/TNS 连接、Oracle Cloud |
| SQL Server | 含 Azure SQL |
| SQLite | 本地文件型 |
| MongoDB | 含 MongoDB Atlas，副本集/分片集群 |
| Redis | 含 Redis 集群、Redis Sentinel、Redis Enterprise Cloud、Microsoft Garnet（17.1 新增） |
| Snowflake | 17 新增的云数仓支持 |

### 1.2 连接方式特点

- **SSH 隧道**：所有主要数据库均可经 SSH 跳板连接；支持密码与私钥（含 PuTTY 格式）认证，Windows 版支持 ssh-agent/Pageant。配置上是连接对话框中独立的 "SSH" 选项卡：勾选启用后填写 SSH 主机、端口、用户名、认证方式，General 选项卡的主机则填数据库相对于 SSH 主机的地址（如 127.0.0.1）。
- **SSL/TLS**：MySQL、PostgreSQL、MariaDB 等支持，独立 "SSL" 选项卡，可配置客户端证书、CA 证书、是否验证服务器证书。
- **HTTP 隧道**：Navicat 特色功能，仅 MySQL、PostgreSQL、SQLite、MariaDB 可用。在 Web 服务器上放置一个 PHP 隧道脚本（ntunnel_mysql.php 等），客户端通过 80/443 端口经该脚本中转访问数据库；支持 base64 编码出站查询以绕过 WAF。**HTTP 隧道与 SSH 隧道互斥。**
- **高级认证**：PAM、LDAP、Kerberos、MFA、SSO。
- **连接测试**：连接对话框内置"测试连接"按钮，保存前即时验证。
- **云数据库兼容**：Amazon RDS/Aurora/Redshift/ElastiCache、Azure、Oracle Cloud、Google Cloud、MongoDB Atlas、Redis Cloud 等。

> 复刻建议：MVP 优先做 MySQL/MariaDB + PostgreSQL + SQLite 直连 + SSH 隧道 + SSL；HTTP 隧道可后置。

## 2. 主界面布局

Navicat 主窗口为经典的"左树 + 顶部工具栏 + 中部对象区/标签页工作区 + 底部信息栏"结构：

### 2.1 顶部主工具栏（Main Toolbar）
一排带文字标签的大图标按钮，是 Navicat 最具辨识度的元素之一：
- **连接（Connection）**：下拉新建各类数据库连接
- **新建查询（New Query）**：打开 SQL 编辑器标签页
- 随后是按对象类型切换的按钮组：**表、视图、函数（存储过程）、用户、其他（事件/触发器/序列等）**
- 工具类按钮：**查询、备份、自动运行、模型、图表（BI）** 等
- 选中对象类型后，工具栏下方会出现**对象工具栏（Object Toolbar）**：如"打开表 / 设计表 / 新建表 / 删除表 / 导入向导 / 导出向导"等针对当前对象类型的操作按钮，外加搜索过滤框。

### 2.2 左侧导航窗格（Navigation Pane / 连接树）
- 树形结构：连接 → 数据库/模式 → 对象分类（表、视图、函数、查询、备份…）→ 具体对象。
- 双击连接打开/关闭连接（打开的连接图标变彩色，关闭为灰色）；双击数据库展开。
- 全部操作均有右键上下文菜单（打开、设计、新建、删除、重命名、转储 SQL、导入/导出、刷新等）。
- 顶部有连接过滤/搜索框。可隐藏整个导航窗格以扩大工作区。

### 2.3 中部对象窗格（Object Pane）+ 标签页式工作区
- 默认显示"对象"标签：当前选中数据库下某类对象的**列表视图**（名称、行数、引擎、修改日期、注释等列），支持列表/详细信息/ER 图三种视图切换（表对象可直接切到 ER 图查看库内表关系）。
- 打开表、查询、设计器等都会在同一区域新增**标签页（Tab）**，标签可拖出为独立窗口。
- 17 新增 **Focus Mode（专注模式）**：隐藏周边面板，仅保留当前编辑区。

### 2.4 信息窗格（Information Pane，下方或右侧）
显示选中对象的详细信息，分子标签：**常规信息（General）、DDL、依赖、成员/权限、项目活动、预览**。可开关显示。

### 2.5 底部状态栏
显示当前连接/数据库上下文、所选对象数量；在数据网格中显示当前记录数、查询耗时、分页导航控件（首页/上一页/下一页/末页、跳页、每页记录数设置）、只读状态、自动提交开关等。

### 2.6 其他全局 UI
- 欢迎/连接管理界面（17 新增集中式连接管理，可批量操作）。
- URI 直达：可生成指向某对象的链接（17 新增）。
- 收藏夹：常用对象/查询快速访问。
- 历史日志窗口：记录实际执行的所有 SQL 命令。

## 3. 核心功能模块清单

### 3.1 连接管理
- 连接分组：物理分组文件夹 + "虚拟分组"。
- **连接颜色标记**：给连接设置颜色，该连接下所有打开的标签页/工具栏区域显示色条，防止误操作生产库。
- 连接测试、复制连接、导入/导出连接配置、密码保存（加密）。
- 17 新增集中式连接管理界面，支持批量编辑多个连接 profile。

### 3.2 表数据网格（Table Viewer / Data Viewer）
- **视图模式**：Grid View（网格）、Form View（表单，单条记录逐字段编辑）、MongoDB 另有 Tree View / JSON View。
- **分页**：默认每页 1000 条（可全局或按表设置 Limit Records），状态栏分页导航。
- **筛选**：可视化 Filter 向导（字段 + 操作符 + 值，组合 AND/OR）；列头快捷筛选。
- **排序**：列头排序箭头，多字段排序。
- **内联编辑**：单元格直接编辑，支持内置编辑器（文本/十六进制/图片/BLOB、JSON 编辑器），新增行、删除行、复制粘贴记录，提交/回滚更改。
- **外键查找（Value Picker）**：编辑外键列时弹出引用表的数据选择面板，可关键字搜索选值；可一键打开外键关联数据。
- **Table Profile（17 新增）**：保存多套"筛选+排序+列显示"组合并快速切换。
- **数据剖析 Data Profiling（17 新增）**：对表数据一键生成分布、类型、空值率等可视化统计。
- 复制为 INSERT/CSV、设置 NULL、原始模式编辑等辅助操作。

### 3.3 表设计器（Table Designer）
多标签结构（标签随数据库类型变化）：
- **字段**：名称、类型、长度、小数点、允许 NULL、主键、默认值、注释、自增、虚拟列等；支持搜索字段、上移下移、插入/删除字段。
- **索引**：索引名、字段（弹出字段选择器，可设长度）、索引类型（NORMAL/UNIQUE/FULLTEXT/SPATIAL）、索引方法（BTREE/HASH）。
- **外键**：名称、字段、被引用模式/表/字段、ON DELETE / ON UPDATE 行为。
- **触发器**：名称、BEFORE/AFTER、INSERT/UPDATE/DELETE、触发器体。
- **唯一键/检查约束/排除约束**（按数据库类型出现）。
- **选项**：MySQL 的引擎、字符集、排序规则、行格式、自增起始值等。
- **注释**：表注释。
- **SQL 预览**：实时生成的 CREATE/ALTER 语句。

### 3.4 SQL 查询编辑器（详见第 4 节）

### 3.5 视图 / 函数 / 存储过程 / 事件管理
- **视图设计器**：定义 SQL + 可视化视图构建器 + 高级属性 + SQL 预览；可直接打开视图查看数据。
- **函数/存储过程设计器**：参数定义向导、返回类型、函数体编辑器、运行按钮带参数输入、结果显示；PL/SQL / PL/PGSQL **调试器**（断点、单步、变量、调用栈）。
- **事件 / 触发器 / 序列 / 同义词 / 物化视图**等对象按数据库类型出现在导航树中。

### 3.6 数据导入导出
- **导入向导**（分步：选格式 → 选文件 → 选项 → 目标表 → 字段映射 → 导入模式 → 执行）：支持 CSV、TXT、Excel、XML、JSON、DBF、Access、HTML 表格、ODBC。导入模式：追加、更新、追加/更新、删除、复制。
- **导出向导**：导出为 CSV、TXT、Excel、XML、JSON、HTML、SQL（INSERT 脚本）、DBF、Access、PDF/RTF 等，可选字段、附加选项。
- **转储 SQL 文件 / 运行 SQL 文件**：单表或整库的结构/数据 SQL 转储与执行（带进度）。
- 导入/导出配置可保存为 profile 供自动运行调用。

### 3.7 数据传输、数据同步、结构同步
- **数据传输**：跨连接、跨库种（异构）把表结构+数据从 A 传到 B，或导出为 SQL 文件；可选对象、自定义高级选项。
- **数据同步**：比对两个库的表数据差异（插入/更新/删除），可视化展示差异行，勾选后生成并部署同步脚本。
- **结构同步**：比对两个库的 DDL 差异，并排显示对象差异和 ALTER 脚本，可勾选部署。

### 3.8 备份 / 还原
- Navicat 自有格式的本地备份/还原（可选对象、压缩），支持定时备份。
- 提取备份中的 SQL。
- 对 MongoDB/Oracle/SQL Server 提供原生备份工具的图形化封装。

### 3.9 用户与权限管理
- 用户列表：新建/编辑/删除用户。
- 用户设计器：常规（用户名、主机、密码）、高级（连接数限制、SSL 要求）、服务器权限、对象级权限（授权矩阵勾选）。
- **权限管理器**：以对象为中心查看/编辑各用户权限。角色管理（PG/Oracle/SQL Server）。

### 3.10 ER 图 / 模型设计器（Model）
- 库内表对象可直接切换 ER 图视图，可视化外键关系。
- 独立模型工作区：正向设计（画 ER 图 → 生成 SQL/同步到库）与逆向工程；概念/逻辑/物理三层模型；17 支持 Relational、Dimensional、Data Vault 2.0。
- 模型与数据库双向同步比对。
- **数据字典（17 新增）**：基于模板生成数据库文档并导出 PDF。

### 3.11 其他模块
- **数据生成**：按规则批量生成测试数据，自动维护外键参照完整性。
- **图表/BI**：数据源 → 数据集 → 拖拽生成图表 → 仪表板。
- **自动运行（计划任务）**：查询、备份、导入导出、数据传输/同步组合成批处理作业，按计划运行，支持邮件通知。
- **历史日志**、**收藏**、**全局搜索**、**暗色主题**。
- **AI 助手（17 新增）**：内置聊天、Ask AI 对 SQL 做解释/优化/格式化/纠错，支持 ChatGPT、Claude、Gemini、Deepseek、Ollama。
- **MongoDB 专属**：Schema 分析器、聚合管道可视化构建。
- **Redis 专属**：键浏览/编辑/搜索、Pub/Sub 实时消息查看。
- **云协作**：同步连接配置、查询、代码片段、模型、BI 工作区给团队。

## 4. 查询编辑器细节

- **语法高亮 + 代码折叠**：关键字/字符串/注释着色，括号匹配，代码折叠。
- **自动补全**：SQL 关键字 + 基于当前连接实时 schema 的对象（库名、表名、列名、函数）；输入 `表名.` 后弹出该表列名；支持手动触发（Ctrl+Space）和模糊匹配；可配置自动弹出与大小写风格。
- **代码片段**：编辑器右侧片段面板，内置 + 自定义片段库，支持占位符；片段进入自动补全候选。
- **SQL 美化**：一键格式化，可配置缩进、关键字大小写。
- **执行控制**：运行全部 / 运行选中片段 / 运行光标所在语句；停止按钮；多条语句顺序执行。
- **多结果集与多标签**：一个查询返回多个结果集时显示多个"结果"子标签；**结果可固定（Pin，17 新增）**。
- **结果网格可编辑**：单表含主键的查询结果可直接增删改；结果可导出。
- **解释计划（Explain）/ 可视化 Explain（17 增强）**：图形化展示执行计划，支持 JSON/文本/统计格式切换。
- **查询构建器**：可视化拖表、勾字段、连线 JOIN、图形化设置 WHERE/GROUP BY/HAVING/ORDER BY，与 SQL 文本双向联动。
- **查询参数**：`$variable` 形式的运行时参数，执行时弹窗输入。
- 其他：查询保存为"查询"对象、查询摘要、文本查找替换、行号、缩放。

## 5. UI 视觉风格

- **整体风格**：Navicat 16 起扁平化设计、线性/双色调扁平图标（主工具栏为"大图标+文字"按钮），浅灰/白为底、品牌色（蓝绿/青色）点缀。17 微调，新增 Focus 模式。
- **暗色模式**：全平台支持，可跟随系统或手动切换；编辑器配色随主题切换。
- **Windows vs macOS 差异**：两端均为原生开发（非 Electron）——
  - macOS 版遵循 Apple HIG：标题栏/工具栏融合、原生控件、系统全屏与触控板手势。
  - Windows 版：菜单栏（文件/编辑/查看/收藏/工具/窗口/帮助）+ 工具栏，对话框密度更高。
  - 功能基本对等，布局细节与快捷键体系（Ctrl vs Cmd）不同。

## 6. 竞品对照（用于圈定 MVP 范围）

| 维度 | Navicat Premium | DBeaver | TablePlus | HeidiSQL |
|---|---|---|---|---|
| 价格 | 商业（最贵） | 开源免费 + PRO | 付费，免费版限 2 标签 | 免费开源 |
| 数据库覆盖 | 9 种主流 + 云 | 最广（80+ JDBC） | 主流关系型 + 少量 NoSQL | MySQL/MariaDB/PG/MSSQL/SQLite |
| 平台 | Win/macOS/Linux 原生 | 跨平台（Java，较重） | 原生、极快、轻量 | 仅 Windows |
| SQL 编辑器 | 强（补全/片段/可视化 Explain/调试器） | 强 | 够用 | 基本 |
| 数据网格编辑 | 强（表单视图、外键选值、Profile） | 强 | 强（inline edit + Cmd+S） | 良 |
| 可视化查询构建器 | 有 | 有 | 无 | 无 |
| ER 图/建模 | 最强 | 有 ER 查看 | 无 | 无 |
| 数据/结构同步 | 强（核心卖点） | PRO 版有 | 无 | 弱 |
| 导入导出 | 格式最全 + 向导 | 全 | 基本 | 良 |
| 备份/计划任务 | 内置 + 调度 + 邮件 | 弱 | 无 | 仅 SQL 导出 |
| BI / 数据生成 / AI | 有 | AI（PRO） | 无 | 无 |

**MVP 分梯队**：
1. **第一梯队（必做）**：连接管理（分组/颜色/测试/SSH/SSL）、左树导航、表数据网格（分页/筛选/排序/内联编辑）、SQL 编辑器（高亮/补全/运行选中/多结果集/结果导出）、表设计器（字段/索引/外键/SQL 预览）、导入导出（CSV/SQL/JSON/Excel）、暗色主题。
2. **第二梯队（差异化）**：可视化查询构建器、结构同步/数据同步、ER 图查看、备份/计划任务、用户权限管理、外键值选择器。
3. **第三梯队（锦上添花）**：BI 图表、数据生成、数据字典、AI 助手、云协作、MongoDB/Redis 专属工具。
